# Skill.md — weekly_reports

Skills and capabilities relevant to working in this project.

## Frontend

- Vanilla HTML/CSS/JavaScript (no framework, no build step)
- Responsive layout with CSS Grid/Flexbox
- DOM manipulation and event handling
- Browser APIs: `localStorage`, `crypto.randomUUID()`, `Blob`, `URL.createObjectURL()`

## Supabase

- Supabase JS client v2 (loaded via CDN)
- CRUD operations: `.select()`, `.upsert()`, `.delete().eq()`
- Error handling from Supabase responses (`{ data, error }`)
- Understanding of Row Level Security (RLS) policies

## Deployment

- Vercel static site deployment
- Configuring root directory in Vercel project settings
- GitHub → Vercel auto-deploy on push to `main`
